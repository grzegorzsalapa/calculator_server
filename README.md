# Calculator server

Simple server based on TCP, calculating well-formed math equations. Intended to use with "calculator_client".

## How to run

#### Installation

    $ pip install .

#### Run from terminal

    $ calculator_server

## How to test

#### Installation

    $ pip install .[test]

#### Run tests

    $ pytest

## Requirements

All packages required to run calculator module are specified in pyproject.toml file.
